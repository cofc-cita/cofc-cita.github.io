The Tools of The Trade

  1. nmap: THE network mapper.
  2. Wireshark: Extremely powerful network packet analysis tool.
  3. hashcat: password cracker if you have a decent GPU
  4. john the ripper: password cracker for computers that have plent of CPU and RAM
  5. OpenVAS: vulnerability scanner for machines on a network
  6. Spunk: data and log analysis tool
  7. Netcat: swiss army knife of network tools
